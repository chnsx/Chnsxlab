在Spring官网中，提供的下一代Spring工具为Spring Tools 4，该工具提供独立的Eclipse版本，也提供插件形式，可集成在Visual Studio Code和Theia中。使用该工具可以进行Spring Boot开发。

Spring Tools 4有许多特性，举例如下：

  • 新手教学
  • 快速代码导航
  • 智能代码补全
  • 实时代码检查
  • 精确的Java Bean信息、报告、配置和细节
  • 开源，可用于商业用途，但需遵守开源许可证

IntelliJ IDEA是一个智能的、上下文感知的IDE，用于在各种应用程序中使用Java和其他JVM语言，如Kotlin、Scala和Groovy。此外，IntelliJ IDEA Ultimate可以帮助你开发全栈式的Web应用，这要归功于它强大的集成工具，对JavaScript和相关技术的支持，以及对Spring、Spring Boot、Jakarta EE、Micronaut、Quarkus、Helidon等流行框架的高级支持。此外，你可以用JetBrains开发的免费插件来扩展IntelliJ IDEA，让你可以使用其他编程语言，包括Go、Python、SQL、Ruby和PHP。

IntelliJ IDEA提供Community和Ultimate两种版本，其中Community版本是开源免费的，可用于商业用途，Ultimate版本提供30天的免费使用期，试用期过后需要购买许可证方能继续使用，未取得许可证的情况下，使用其从事商业和盈利用途，您将会收到律师函。

Q：我使用Windows7系统，可以安装IDEA吗？

最新版IDEA不支持Windows7系统，您可以安装历史版本，如果您不知道应安装哪种版本，请安装Toolbox App，它会帮您选择。

Q：我的计算机上没有安装课程中提到的四种浏览器中的任何一个，我使用其它浏览器，可以吗？

当然，如果使用其它浏览器，请确保为最新版本。之所以作为先决条件，是因为如果要真正从事Web开发，应选用它们，它们包含最新的技术支持，这可以避免许多麻烦。

Q：我选择安装IntelliJ IDEA Community可以吗？

Community版本虽然为开源软件，可免费试用，但仅支持纯Java开发，不提供对Java Web的支持，换言之，不提供对Spring和Spring Boot的支持，请安装Ultimate版本，它包含对Spring的全方位支持。

Q：我是Windows用户，安装软件时，我不知道如何选择安装路径？

推荐您选择非系统盘下的根目录Program Files或Program Files (x86)，在里面创建名为您的软件名称的子文件夹，然后选择此文件夹作为安装路径。

Q：我通过特殊手段获得了IDEA的免费使用权，可以吗？

勿以善小而不为。请支持正版软件，打击非法侵害互联网软件权益的违法行为。

Q：我选择安装Spring Tools 4，下载完成后点击*.jar弹出选择打开程序对话框，如何解决？

这是因为您的计算机中没有Java环境，有两种解决办法：

  1. 安装Java环境(jdk)。
  2. 将其作为压缩包使用解压软件打开，找到contents.zip文件，将此文件解压出来即可。

Q：我安装了IDEA或Spring Tools 4，但是Java在哪？我需要自己额外下载Java环境吗？

您只有在使用Visual Studio Code进行Java开发时才需要额外下载Java环境，IDEA和Spring Tools 4 for Eclipse已经内置了Java环境。

Q：我从事过Java Web开发，请问maven、tomcat这些工具需要额外安装吗？

您可能没有使用Spring Boot开发过Java Web程序，这些工具都已经内置在IDEA或Spring Tools 4中。

Q：我应该如何选择IDE？

本套课程选用IDEA作为教学开发环境IDE。如果您对Eclipes或者Visual Studio Code情有独钟，请选择属于您的IDE.

Q：IDE和IDEA有什么关系？

IDE(Integrated Development Environment)全称为集成开发环境，是用于提供程序开发环境的应用程序，一般包括代码编辑器、编译器、调试器和图形用户界面等工具。集成了代码编写功能、分析功能、编译功能、调试功能等一体化的开发软件服务套。所有具备这一特性的软件或者软件套（组）都可以叫集成开发环境。如微软的Visual Studio系列，Borland的C++ Builder、Delphi系列、Jetbrain的IDEA等。该程序可以独立运行，也可以和其它程序并用，大大方便了开发者。
